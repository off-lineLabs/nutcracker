Serviços de Terceiros

No entanto, não podemos garantir que as funcionalidades de terceiros usadas pelo aplicativo se comportem da mesma maneira. Os seguintes serviços de terceiros podem coletar dados técnicos limitados:

6.1. Google Play Services (Relatório de Falhas)

  • Coleta automaticamente dados de falhas e diagnósticos para ajudar a identificar e corrigir bugs
  • Pode coletar: modelo do dispositivo, versão do Android, logs de falhas, versão do aplicativo, identificador de instalação anônimo, localização aproximada (nível de país), timestamps
  • Isso é obrigatório para qualquer aplicativo publicado na Google Play Store e não pode ser desabilitado
  • Acessamos relatórios de falhas apenas para corrigir bugs e melhorar a estabilidade do aplicativo
  • Política de Privacidade: https://policies.google.com/privacy

6.2. ML Kit Barcode Scanning (Google)

  • Processa imagens de código de barras para escanear códigos de produtos
  • O escaneamento de código de barras é realizado no dispositivo quando possível
  • Pode enviar dados de telemetria limitados para o Google
  • Política de Privacidade: https://policies.google.com/privacy

6.3. API Open Food Facts

  • Fornece informações nutricionais sobre produtos alimentícios
  • As solicitações são feitas diretamente do seu dispositivo para seus servidores
  • Pode coletar: seu endereço IP, códigos de produtos solicitados, timestamps
  • Open Food Facts é um projeto sem fins lucrativos e de código aberto
  • Política de Privacidade: https://world.openfoodfacts.org/privacy-policy

6.4. Base de Dados de Exercícios Gratuita (GitHub)

  • Fornece demonstrações e informações de exercícios
  • As solicitações são feitas diretamente do seu dispositivo para os servidores do GitHub
  • Pode coletar: seu endereço IP, timestamps
  • Política de Privacidade: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement
